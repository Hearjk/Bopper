Installation Guide

1

Download & Unzip

Click the Download button and unzip Bopper.zip

2

Copy Plugin

Move Bopper.component to:

~/Library/Audio/Plug-Ins/Components/

Tip: Press Cmd+Shift+G in Finder and paste the path

3

Allow Plugin (Required)

Open Terminal and run this command:

xattr -cr ~/Library/Audio/Plug-Ins/Components/Bopper.component

This removes macOS quarantine since the plugin isn't notarized

4

Restart DAW

Restart Logic Pro, Ableton, or your DAW. Bopper will appear as an AU instrument!
